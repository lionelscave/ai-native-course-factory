# Course App Integration Contract

The course app should consume:

- `course/course-manifest.json`
- module markdown in `course/modules/`
- `beautiful-ai/deck-manifest.json`
- `revenue/pricing-plan.md`
- `marketing/landing-page-copy.md`

## App requirements

- Course landing page
- Readiness assessment
- Module browser
- Beautiful.ai deck links/embeds
- Lead capture / waitlist
- Offer tier CTA
- Student progress state
- Admin content manifest reload
