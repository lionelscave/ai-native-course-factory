# Module 03 — Control Plane Architecture

## Executive outcome
Learners design the control plane that governs agents, tools, memory, models, policies, approvals, and economics.

## Lessons
1. Control plane vs application layer
2. Agent identity, tool access, and data boundaries
3. Memory, traceability, and observability
4. Runtime policy and kill-switch patterns
5. Economic controls and model routing

## Artifact
Control-plane blueprint.

## Exercise
Design a control-plane map for one business workflow.
